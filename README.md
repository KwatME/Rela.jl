Module for relating elements in medicine.

## Install

```sh
julia
```

```julia
using Pkg: add

add(url="https://github.com/KwatME/Rela.jl")
```

## Use

See [examples](notebook/example.ipynb).

# " (comment)

# { (contain)

# > (become)

# + (make)

# - (break)

# / (increase amount)

# \ (decrease amount)

# ^ (increase function)

# v (decrease function)
